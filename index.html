Console d'administration

Systeme d'agents specialises - etat reel des depots, lu a l'ouverture. Cette page ne contient aucune donnee : tout est lu en direct avec ta cle.

Parcours : Zone 1 (triage, informations-publiques/) - Zone 2 (docs accessibles, connaissances-stockees/) - Zone 3 (memoire vive, savoir/). La pastille pleine est la zone ou vit le document. Statut de l'info : Etabli / Annonce / Non verifie.

0 document selectionne

Fraicheur de la memoire vive de chaque agent : le numero de savoir/VERSION.md au depot et sa derniere evolution. Un agent sans fichier de version n'a pas encore de protocole memoire.

Une ligne par veille declaree active dans ADMIN/instructions/VEILLES_ACTIVES.md. Age = jours ecoules depuis la derniere entree de l'index de veille. Au-dela de 8 jours, la veille n'a pas rendu compte : ecart a traiter.

Le depot est la verite ; cette console n'est qu'une vue. Depot public : il ne contient que le code de cette page - aucun document, aucun resume, aucune cle.
